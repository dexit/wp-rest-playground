A playground for WP REST API in the admin area.

== Description ==

This plugin creates a page unders tools which loads WP REST API Backbone.js client library.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Tools->WP REST Playground and open developer tools to play with the Backbone.js client library.

== Frequently Asked Questions ==

= Does this plugin add a REST Endpoint

No, It does not.
